🍎 Mac環境でDockerを使う

MacでDockerを使うには、Docker Desktop for Macのインストールが必要です。

📋 インストールするもの

  • Docker Desktop - Dockerをグラフィカルに管理できるツール

MacではWindowsのようにWSL2は不要で、Docker Desktopだけでシンプルに使えます。

🐳 Docker Desktop for Mac のインストール

📥 ダウンロード

公式サイト

https://www.docker.com/products/docker-desktop/

⚠️ Mac チップ別選択

ダウンロードする前に、お使いのMacのチップを確認してください：

  • Apple Silicon（M1/M2/M3/M4/M5など）: 「Apple Chip」版を選択
  • Intel Mac: 「Intel Chip」版を選択

💡 確認方法：画面左上のAppleメニュー → 「このMacについて」→ プロセッサの項目を確認

📝 インストール手順

1 お使いのMacに合ったバージョンをダウンロード

.dmgファイルがダウンロードされます

2 .dmgファイルを開く
3 Docker.appをアプリケーションフォルダにドラッグ
4 アプリケーションフォルダからDocker Desktopを起動
5 利用規約に同意

初回起動時に表示されます

✅ 確認方法

ターミナル（Terminal.app）で以下のコマンドを実行:

docker --version
docker compose version

🎉 インストール成功

バージョン情報が表示されれば、Docker Desktopのインストール成功です！

💡 補足

Docker Desktopが起動していないとコマンドが使えません。メニューバーにDockerアイコンが表示されていることを確認してください。

🔧 よくあるトラブルと対処法

❌ トラブル1: Docker未起動

エラー: "Cannot connect to the Docker daemon"

解決策: アプリケーションフォルダからDocker Desktopを起動します。メニューバーにDockerアイコンが表示されていることを確認してください。

❌ トラブル2: 間違ったチップ版をダウンロード

症状: インストールできない、または動作が遅い

解決策: お使いのMacのチップに合ったバージョンを再ダウンロードしてください。Apple SiliconとIntel版は互換性がありません。
