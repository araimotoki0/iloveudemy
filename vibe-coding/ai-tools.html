難易度マップ

上が易しく、下にいくほど難しいがAIの自由度も高くなります。

EASY
🐙
GitHub Copilot 初級
勝手に続きを書いてくれる
コード補完特化 → 出力が最も予測しやすい、でも狭い
動く場所 IDE内（VS Code等）
誰向け エンジニア
自律度 AIが次の行を予測・補完
リスク ほぼなし
💡 使い方：コードを書いていると次の行を自動補完。Tabキーで採用するだけ。
⌨️
Cursor 初級
チャットしながらコードを直す
コーディング特化 → 出力が予測しやすい、でも狭い
動く場所 IDE内（ローカル）
誰向け エンジニア
自律度 人間が主導、AIがチャットで補助
リスク 低い
💡 使い方：サイドチャットで「この関数リファクタして」等を依頼。差分を確認して承認。
💬
ChatGPT / Claude.ai 初級
なんでも聞ける賢い相談相手
汎用チャット → 出力が予測しやすい、幅広いが手動
動く場所 ブラウザ / アプリ
誰向け 誰でも
自律度 質問→回答の一問一答
リスク ほぼなし
💡 使い方：ブラウザで質問・文章作成・翻訳・壁打ち。ファイルを添付して分析も。
🤖
Manus 初〜中級
指示だけ出せばクラウドで勝手に動く
汎用エージェント → 結果は比較的安定、広い
動く場所 ブラウザから指示 → クラウドで実行
誰向け 誰でも（非エンジニアOK）
自律度 AIが自律実行、並列タスクも可
リスク 低い（自分のPCに触れない）
💡 使い方：ブラウザから「競合10社のリサーチまとめて」等を指示。クラウド上でAIがブラウザ操作・資料作成まで自律完了。自分のPCは自由に使える。
👨‍💻
Devin 中級
ジュニアエンジニアのAI版
ソフトウェア開発エージェント → 自律型
動く場所 クラウド（専用環境）
誰向け エンジニア・開発チーム
自律度 Issue → PR作成まで自律で完了
リスク 低〜中（コードレビュー必須）
💡 使い方：GitHubのIssueを渡すとコード書いてPR作成。Slackで進捗報告してくれる。
🔧
Claude Code 中〜上級
万能ナイフ、でも自分で研ぐ
柔軟な設計 → 自由度が高い分、結果がブレやすい
動く場所 CLI（ローカルPC）
誰向け エンジニア〜パワーユーザー
自律度 エージェント的に動けるが自前構築
リスク 中（権限範囲次第）
💡 使い方：ターミナルから「このリポジトリのテスト全部書いて」等。ファイル操作・Git操作も可。
HARD
🦞
OpenClaw 上級
PCごとAIに渡す。専用機必須。
PC丸ごと委任 → 最も自由だが結果の予測が難しい
動く場所 ローカルPC全体（専用機推奨）
誰向け パワーユーザー・開発者
自律度 24時間自律稼働、スキル自己獲得
リスク 高い（全データ・全権限を開放）
💡 使い方：LINEやSlackから指示。メール管理、飛行機予約、コード修正、SNS運用など何でも自律実行。

どれを選ぶ？

🗺️ 目的別ガイド
●　コードを書くだけ → Copilot / Cursor
●　質問・文章作成・相談 → ChatGPT / Claude.ai
●　非エンジニアで長いタスクを丸投げ → Manus
●　開発タスクを自律で回したい → Devin
●　自分で環境を組める＋柔軟に使いたい → Claude Code
●　全部AIに任せる覚悟がある → OpenClaw（専用PC推奨）
