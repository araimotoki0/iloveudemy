Claude Codeとは何だったのか

Claude Codeは2025年2月24日にAnthropicがリサーチプレビューとして公開した、ターミナルベースのAIコーディングアシスタントです。当初は「ターミナルでチャットしてファイルを編集し、Bashコマンドを実行できる」シンプルなツールでした。

💡 ChatGPTより速い収益成長

Claude CodeはGA（一般提供）開始からわずか約6ヶ月で年間10億ドル（約1500億円）の年間売上ペースに到達しました。これはChatGPTの収益成長ペースを上回る驚異的なスピードです。

2025年前半：誕生と成長

2025年2月24日
Claude Code リサーチプレビュー公開 v0.2.x Claude 3.7 Sonnet

AnthropicがClaude 3.7 Sonnetと同時にClaude Codeを発表。ターミナルで動作するAIコーディングアシスタントとして誕生。

  • ターミナルでの対話型コーディング
  • ファイルの読み書き・編集
  • Bashコマンド実行
  • CLAUDE.mdによるプロジェクト指示
  • MCP（Model Context Protocol）サポート
2025年3月〜4月
急速な改善とバグ修正
  • 権限システムの改善
  • カスタムスラッシュコマンド（.claude/commands/）の追加
  • パフォーマンスの最適化
  • git統合の強化
2025年5月
一般提供（GA）開始 Sonnet 4 / Opus 4

Claude Codeが一般提供（GA = General Availability）に移行。Sonnet 4とOpus 4モデルが利用可能に。

  • 誰でもインストール・利用可能に
  • 安定版としてのリリース
  • 企業向け機能の充実
2025年6月〜7月
エンタープライズ採用の急増

企業での採用が加速。Claude Code収益が5.5倍に成長。

  • Hooksシステムの導入
  • サブエージェント（Task tool）機能
  • Planモード（Shift+Tab）
  • Amazon Bedrock / Google Vertex AI 対応

2025年後半：プラットフォーム化

2025年8月
Claude for Chrome 公開

Google Chrome拡張機能としてClaude for Chromeを公開。ブラウザからClaude Codeの機能を利用可能に。

2025年9月〜10月
マルチプラットフォーム展開 v1.0.x Opus 4.1

Claude Codeがターミナル以外にも拡大。

  • Claude Code Web版の公開
  • iOS アプリ対応
  • Opus 4.1モデルの導入
  • /contextコマンドの追加
2025年11月
v2.0 - 大型リニューアル v2.0.0

Claude Codeの大型メジャーアップデート。

  • VS Code拡張機能の新ネイティブ版公開
  • UIの完全リニューアル
  • /rewind コマンド追加
  • /usage 使用量追跡
  • Tabキーで思考モード切り替え
  • Ctrl+Rで履歴検索
2025年12月
バックグラウンドエージェントと名前つきセッション Opus 4.5

Opus 4.5モデルの公開と、Claude Code for Desktop の登場。

  • バックグラウンドエージェント - 裏で並行作業
  • 名前つきセッション（/rename）
  • プロンプトサジェスト機能
  • モデル切り替え（Alt+P）
  • Pro向け追加使用量購入

2026年：次世代へ

2026年1月
v2.1 - Skillsシステムと大幅機能追加 v2.1.0

Claude Codeの拡張性が大幅に向上。

  • SKILL.mdによるスキルシステム
  • セッションフォーク機能
  • クラウドハンドオフ（&演算子）
  • --from-pr フラグ
  • 矢印キーによる履歴操作
  • JetBrains プラグイン
2026年1月末
Agent Teams（エージェントチーム） v2.1.32 Opus 4.6

マルチエージェント協調の新時代。Claude Opus 4.6リリース。

  • 1Mトークンコンテキストウィンドウ（ベータ）
  • Agent Teams - 複数エージェントが協調作業
  • チームリードとタスクリストの概念
  • Pluginシステムの導入
2026年2月
安定性の向上と新機能追加 v2.1.45+ Sonnet 4.6
  • Claude Sonnet 4.6 対応
  • Remote Control - スマホからセッション操作
  • ConfigChange フックイベント追加
  • VS Codeプランプレビューの改善
  • メモリ使用量の最適化
  • Windows対応の大幅改善
  • 自動メモリ（auto-memory）機能
  • /copy コマンド追加

モデルの変遷

Claude Codeで使用できるモデルは急速に進化してきました。

2025年2月

2025年5月（GA）
Claude Sonnet 4 / Opus 4
Claude 4世代。GA開始と同時にリリース。
2025年秋
Claude Opus 4.1 → Opus 4.5

2026年1月（現在）
Claude Opus 4.6 / Sonnet 4.6
1Mトークンコンテキスト。最高性能のフロンティアモデル。
