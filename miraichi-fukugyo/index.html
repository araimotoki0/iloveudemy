辞めるときに「倍以上出すから残って」と引き留められた

この副業は4ヶ月ほど継続しました。 ですが、本業であるSIerの開発業務が多忙になってきたため、退職することにしました。

「給料を倍以上の8万円にするから、残ってほしい」

辞めると伝えたとき、こう引き留められました。 正直「もっと早く言ってほしかった」と思いましたが、 本業との兼ね合いでどうしても無理だったため、丁重にお断りしました。

なぜ副業の話を始めようと思ったか

最後に、今回のセミナーの本題でもある社内研修のご案内です。 冒頭で「スクール」という言い方をしたかもしれませんが、 正確にはスクールではなく社内研修。 これから2人のメンバーが受ける予定の社内研修に、 皆さんも一緒に受けませんか、というものです。

なぜこの研修をやろうと思ったのか。きっかけは、今まさに進めているある案件でした。

実はその案件、非プログラマーの方に一人手伝ってもらっています。 Claude Code を使い、ほぼ自分では実装せず、 AIが暴走しないようガードレールを組んだ中で開発してもらうという形です。

🧑‍💻
STEP 1　開発
非プログラマーの方が、ガードレールの中で Claude Code を使って実装する。
⬆️
STEP 2　プッシュ
コードが書けたら、GitHub にプッシュしてもらう。
✅
STEP 3　レビュー
私が安全に実装されているか・要件が間違っていないかを確認してレビュー。

現状、このやり方でうまく回っています。 その方を採用した理由は、難しいスキルではなく——

🙆 採用した理由

  • 必ずやりきってくれるという安心感
  • 諦めない気持ち
  • 時間があったこと
🚀 AIがある今の時代だからこそ

「やりきろう」という気持ちさえあれば、副業は非常にやりやすくなっています。
昔のように1年・2年という長い修行期間がなくても、すぐに始められる。
その中で経験と信頼を積み重ねていけば、 副業自体も少しずつ大きくしていけるはずです。
